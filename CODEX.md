# LOGENESIS-1.5 – Codex Working Contract

## System Intent
This repository implements the Logenesis Engine: a reasoning-first AI
architecture focused on intent validation, ethical constraint checking,
and process-supervised cognition.

Codex should prioritize:
- correctness over speed
- explicit reasoning over shortcuts
- architectural consistency (Inspira / Firma duality)

## What Codex MAY do
- Refactor code for clarity and modularity
- Add tests for reasoning integrity
- Optimize async / event-driven logic
- Propose improvements with explanation

## What Codex MUST NOT do
- Remove ethical checks
- Collapse Inspira into Firma
- Replace reasoning with heuristic shortcuts
- Introduce opaque magic behavior

## Key Concepts Codex must respect
- Inspira = intent / will / ethics
- Firma = feasibility / constraints / execution
- Checker = constitutional boundary
- Learning ≠ memory dump (weighted resonance only)

## Preferred Style
- Explicit logic
- Type hints
- Docstrings explaining WHY, not just WHAT
